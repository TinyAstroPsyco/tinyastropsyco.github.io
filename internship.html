Systems Engineering Internship — What I Did

Fulfil Solutions, CA  •  May 2024 – Aug 2024
Focus areas: Robotic System Engineering, Computer Vision, Reliability Testing

1. System Bring-Up & Calibration

Situation
The production system needed reliable, repeatable motion on all axes so it could run automated cycles without crashing, drifting, or requiring manual babysitting.
Task
Get the hardware to a "production ready" state by setting motion limits, tuning motors, and verifying positional accuracy.
Action
Calibrated endpoints and soft limits, tuned motor parameters (velocity, accel, current limits), aligned sensors, and helped debug misalignment issues on real hardware. I also documented the calibration flow so other engineers/techs could repeat it.
Result
The system was able to consistently hit target positions with reduced overshoot and less operator intervention. In other words: it went from "lab state" to "we can actually run this."

2. Motor & Gearbox Thermal Study

Situation
Motors and gearboxes had to survive the required duty cycle without overheating or wearing out too fast.
Task
Validate whether the current motor + gearbox setup could safely handle production loads over time.
Action
Instrumented the motors/gearboxes, collected temperature data under different loads and run profiles, and analyzed how fast each assembly heated up versus safe operating limits.
Result
Flagged thermal risk areas early and provided guidance on safe usage ranges. This helped prevent overheating, reduced chances of unplanned downtime, and protected hardware life.

3. Vision-Based Auto-Calibration Prototype

Situation
The system used proximity sensors to detect a reference tab on tall storage stacks and align to it. But this method only gave mm-level accuracy. Even a few millimeters off caused jams, scraping, and collision when placing trays into the L-shaped slots.
Task
Improve alignment accuracy so the robot could pick and place trays into vertical stacks without mechanical interference.
Action
Prototyped a camera/vision-based auto-calibration flow. Instead of trusting the proximity sensor reading, the system visually localized the tab/feature, computed a more accurate offset, and fed that corrected position back into motion.
Result
Reduced stack misalignment and lowered jam / collision risk during tray insertion and retrieval. This directly increases reliability and uptime of the storage module.
