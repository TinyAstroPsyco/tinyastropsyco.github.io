Robotics Engineer — Full-time Experience

Eternal Robotics Pvt Ltd, Hyderabad, India  •  Aug 2021 – Jun 2023
Focus areas: Autonomous Systems, Motion Planning, Embedded Systems Development

1. Autonomous Robotic Platforms Development

Situation
Eternal Robotics needed to develop two autonomous robotic platforms for industrial applications: a wall-painting robot and a lane-marking robot. Both required reliable autonomous operation in real-world environments.
Task
Lead the development efforts for both platforms, ensuring they could operate autonomously with feedback integration from both simulation and real-world testing environments.
Action
Designed and implemented control systems for both platforms, created simulation environments for testing, conducted extensive field trials, and iteratively refined the systems based on performance data. Coordinated between simulation results and real-world deployment to ensure robust operation.
Result
Successfully deployed two autonomous robotic platforms that operated reliably in industrial settings. Both robots demonstrated consistent performance with minimal human intervention required.

2. 6-DOF Robotic Arm Motion Planning

Situation
The wall-painting robot required a 6-DOF robotic arm capable of painting complex wall surfaces with precision. Standard motion planning approaches weren't sufficient for the smooth, continuous trajectories needed for high-quality painting.
Task
Develop and deploy manipulation and Cartesian motion planning strategies that enable precise and smooth end-effector trajectories over complex wall surfaces with repeatable performance.
Action
Implemented inverse kinematics solvers and trajectory optimization algorithms. Integrated these into Gazebo for simulation testing before deployment. Developed Cartesian space planning to ensure smooth paint application, optimizing for both speed and paint quality. Conducted extensive testing to validate trajectory smoothness and repeatability.
Result
Achieved precise, smooth end-effector motion that resulted in high-quality paint coverage. The system demonstrated repeatable performance across different wall surfaces and configurations, meeting industrial quality standards.

3. Custom Data Acquisition System

Situation
There was a critical need to validate simulation results against real-world robot performance. Existing data logging solutions were either too expensive or didn't support the required multi-sensor configuration.
Task
Design and build a custom data acquisition device capable of logging data from multiple sensors simultaneously to enable verification of simulation predictions.
Action
Developed a custom ESP32-based data acquisition system supporting multi-sensor data logging via SPI interface. Implemented efficient data buffering and storage protocols. Created accompanying software for data analysis and comparison with simulation outputs.
Result
Successfully validated simulation models against real-world performance, identifying and correcting discrepancies. The device became a standard tool for system validation, significantly reducing development iteration time.

4. Jerk-Limited Motion Profile Generator

Situation
The two-stage actuator controlling spray distance experienced excessive jerk during motion transitions, causing instability in the robot platform and degraded paint quality due to uneven spray patterns.
Task
Develop an online motion profile generator that dynamically tunes motion profiles based on spray distance requirements while minimizing jerk to improve both robot stability and paint quality.
Action
Created an online jerk-limited curve generator that computed optimal motion profiles in real-time. Implemented dynamic tuning based on current spray distance and target position. Integrated feedback loops to continuously optimize performance based on actual system behavior.
Result
Significantly reduced jerk during actuator motion, improving robot stability and paint application quality. The dynamic tuning approach allowed the system to adapt to varying operational conditions, resulting in consistent performance across different painting scenarios.
